# 🦠 COVID-19 Data Analysis & Visualization Dashboard

## 📌 Overview

This project is a **COVID-19 Data Analysis Dashboard** built using **Python and Streamlit**. It provides interactive visualizations and insights into global COVID-19 trends, including confirmed cases, deaths, and recoveries across different countries.

The goal of this project is to help users **understand pandemic patterns** through data-driven analysis and simple visual representation.

---

## 🚀 Features

* 📊 Interactive dashboard using **Streamlit**
* 🌍 Country-wise COVID-19 analysis
* 📈 Time-series visualization of confirmed cases
* ⚰️ Analysis of deaths and recoveries
* 🔍 Data filtering and exploration
* 🤖 Basic prediction using **Linear Regression (ML model)**

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit** (for dashboard UI)
* **Pandas & NumPy** (data processing)
* **Matplotlib & Seaborn** (visualization)
* **Scikit-learn** (machine learning model)

## ▶️ How to Run

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. Navigate to the project folder

   ```bash
   cd your-repo-name
   ```

3. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

4. Run the Streamlit app

   ```bash
   streamlit run app/app.py
   ```
